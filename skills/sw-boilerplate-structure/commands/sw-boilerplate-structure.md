# /sw-boilerplate-structure

Creates the source code skeleton directly in the workspace root based on `requirements/tech-stack.yaml`.

## Usage

```
/sw-boilerplate-structure
```

## Creates

Entry points and module stubs: `src/main.ts`, `app/layout.tsx`, `Application.java`, `main.py`, `cmd/main.go`, and all supporting files.

## Requires

`requirements/tech-stack.yaml` (run `/sw-tech-stack-planner` first)
