# /sw-boilerplate-docker

Creates Docker infrastructure files directly in the workspace root based on `requirements/tech-stack.yaml`.

## Usage

```
/sw-boilerplate-docker
```

## Creates

`docker-compose.yml`, `Dockerfile`, `Dockerfile.frontend`, `.dockerignore`, and `traefik.yml` if applicable.

## Requires

`requirements/tech-stack.yaml` (run `/sw-tech-stack-planner` first)
