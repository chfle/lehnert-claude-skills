# /sw-boilerplate-ui

Creates UI foundation files directly in the workspace root based on `requirements/tech-stack.yaml`.

## Usage

```
/sw-boilerplate-ui
```

## Creates

`tailwind.config.ts`, `globals.css`, `components.json` (shadcn/ui), `lib/utils.ts`, `App.tsx`, or Vue app files – based on detected UI library.

## Requires

`requirements/tech-stack.yaml` (run `/sw-tech-stack-planner` first)
